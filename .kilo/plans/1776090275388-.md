# In _ocr_infobox_with_retries(), when ocr_attempt > 0:

    window_bgr = capture_region((win_left, win_top, win_width, win_height))

## Priority 2: OCR & Matching Improvements (High Impact)

### OCR-01: Roman Numeral Tier Suffix Correction
**File:** `src/autoscrapper/core/item_actions.py`  
**Issue:** Tesseract misreads Roman numerals: `IVŌåÆ1V`, `IIIŌåÆ111`, `IIŌåÆ11`, `IŌåÆ1`.

**Implementation:**
```python
OCR_ALIASES: dict[str, str] = {
    " 1v": " iv",
    " 111": " iii",
    " 11": " ii",
    " 1": " i",
}

def _fix_roman_ocr_suffix(name: str) -> str:
    _ROMAN_ONLY = re.compile(r"^[ivx]+$", re.ASCII)
    for bad, good in OCR_ALIASES.items():
        if name.endswith(bad):
            corrected_suffix = good.strip()
            if _ROMAN_ONLY.match(corrected_suffix):
                return name[: -len(bad)] + good
    return name
```

## Priority 4: Performance Optimizations

### PERF-01: Lazy Load Heavy Dependencies
**Current:** All imports loaded at module import time  
**Target:** `tesserocr`, `cv2`, `numpy` can be deferred until first use

**Implementation:**
```python
def _get_cv2():
    global _cv2
    if _cv2 is None:
        import cv2 as _cv2
    return _cv2
```
