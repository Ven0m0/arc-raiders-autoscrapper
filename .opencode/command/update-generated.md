# Update Generated Data

Regenerate snapshot data and default rules.

**Command:** `uv run python scripts/update_snapshot_and_defaults.py`

**Dry-run:** `uv run python scripts/update_snapshot_and_defaults.py --dry-run`

**Warning:** Do not hand-edit generated outputs in `src/autoscrapper/progress/data/` or `src/autoscrapper/items/items_rules.default.json`.
